<h1>Opis Projektu</h1>
To jest prosty projekt strony internetowej wykonany wyłącznie przy użyciu HTML i CSS. 
Projekt wykorzystuje design dostępny na platformie <a href="https://www.freepik.com/free-vector/log-landing-page-with-colorful-flat-landscape_5414259.htm#query=website%20login%20page&position=26&from_view=keyword&track=ais&uuid=cc91f1e8-0531-4c01-a4c9-7c2019d953eb">Freepik</a>. 
Strona zawiera formularz logowania oraz krajobraz z górami, chmurami i balonami,
które zostały stworzone za pomocą CSS. Należy zauważyć, że strona nie jest responsywna i została zaprojektowana 
wyłącznie dla ekranów o określonej rozdzielczości.
